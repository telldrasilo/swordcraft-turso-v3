# Примеры и черновики

Каталог **не является частью** production-приложения (`src/`). Скрипты здесь не входят в `npm run test` и `npm run build`.

- **websocket/** — демо WebSocket.
- **loot-usage.ts** — исторический пример; пути к `../lib/loot-generator` могут не совпадать с деревом репозитория (см. `docs/Random_gen/`).

Проверка типов для примеров — опционально: `tsconfig.examples.json` (при необходимости поправьте импорты под актуальную раскладку).
