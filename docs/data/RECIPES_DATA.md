# Данные рецептов

## Обзор
Рецепты определяют, какое оружие можно создать, какие части у него есть и какие базовые параметры оно получает до применения материалов и техник.

## Главные файлы
- `src/data/weapon-recipes.ts`
- `src/data/recipes/`
- `src/data/refining-recipes.ts`
- `src/types/craft.ts`
- `src/types/craft-v2.ts`

## Рецепты оружия
Рецепт обычно задает:
- `id`
- имя
- тип оружия
- tier
- базовые статы
- состав частей
- требования к материалам
- ограничения по прогрессу

## Части рецепта
Для v2 особенно важны parts. Они определяют:
- какие категории материалов можно ставить
- сколько материала нужно
- какие свойства важнее при расчете

## Рецепты переплавки
`src/data/refining-recipes.ts` описывает преобразования ресурсов, например:
- руда -> слитки
- металлы -> сплавы
- сырье -> обработанный материал
