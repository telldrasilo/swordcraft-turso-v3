# Данные техник

## Обзор
Техники — это модификаторы процесса крафта, которые меняют качество, характеристики, надежность, стоимость или этапы изготовления.

## Главные файлы
- `src/data/techniques/basic.ts`
- `src/data/techniques/advanced.ts`
- `src/types/craft-v2.ts`
- `src/lib/craft/calculator.ts`

## Группы техник
В проекте есть базовые и продвинутые техники.

Они могут влиять на:
- качество
- атаку
- прочность
- скорость
- точность прогноза
- расход материалов
- состав этапов

## Роль в крафте
Техника не существует отдельно: она применяется поверх рецепта и материалов.

**Крафтовая линия и микрозадачи:** целевой UX процесса крафта — полоса **этапов** (блоки = применяемые техники) и **микроэтапов** (подсегменты = микрозадачи в данных техники); порядок техник (плавка/обработка раньше сборки и приёмов ковки), доли времени и сообщения из микрозадач — см. **[ENCYCLOPEDIA_MATERIALS_TECHNIQUES_ROADMAP.md](../ENCYCLOPEDIA_MATERIALS_TECHNIQUES_ROADMAP.md)** **§10**, **§12**. Боевые техники из этого файла — один из типов блоков на линии.

Из-за этого при изменении техники нужно проверять:
- итоговый расчет оружия
- прогноз
- UI выбора техник
- требования к разблокировке
