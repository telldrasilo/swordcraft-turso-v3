# Данные техник

## Обзор
Техники — это модификаторы процесса крафта, которые меняют качество, характеристики, надежность, стоимость или этапы изготовления.

## Главные файлы
- `src/data/techniques/basic.ts`
- `src/data/techniques/advanced.ts`
- `src/types/craft-v2.ts`
- `src/lib/craft/calculator.ts`

## Группы техник
В проекте есть базовые и продвинутые техники.

Они могут влиять на:
- качество
- атаку
- прочность
- скорость
- точность прогноза
- расход материалов
- состав этапов

## Роль в крафте
Техника не существует отдельно: она применяется поверх рецепта и материалов.

Из-за этого при изменении техники нужно проверять:
- итоговый расчет оружия
- прогноз
- UI выбора техник
- требования к разблокировке

## Что смотреть при изменениях
- `src/data/techniques/basic.ts`
- `src/data/techniques/advanced.ts`
- `src/lib/craft/calculator.ts`
