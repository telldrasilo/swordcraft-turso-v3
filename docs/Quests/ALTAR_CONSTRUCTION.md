# Постройка алтаря зачарований (полное руководство по фазам)

## Общая механика

Вкладка **«Алтарь»** в кузнице содержит 5 последовательных фаз строительства.  
Каждая фаза требует определённые **материалы** (со склада) и **изученные техники**.  
Когда требования выполнены, становится доступна кнопка **«Начать строительство»** – запускается **многоступенчатый крафт** (серия этапов с таймерами).  
После завершения всех этапов фазы появляется чекбокс **«Фаза X: завершена»**, и открывается следующая фаза.  

> **Квестовые артефакты** (резонаторная матрица, фокусирующая чаша, лунный камертон) не расходуются, а **встраиваются** в алтарь во время фазы III. Они должны быть в инвентаре для старта фазы III.

После завершения **фазы V** вкладка «Алтарь» заменяется на **«Улучшения алтаря»** (пока плейсхолдер «В разработке»).

# Примечания для реализации

- **Прогресс каждой фазы** сохраняется в `gameStore.altarPhasesCompleted` (массив boolean для 5 фаз).  
- **Текущий активный крафт** для фазы хранится в `ActiveCraftV2` (переиспользовать систему очереди этапов).  
- **Отмена строительства** внутри фазы: возвращаются материалы за ещё не пройденные этапы; завершённые этапы остаются пройденными. Для фазы III: если отмена до завершения всех этапов – квестовые артефакты возвращаются в инвентарь. После завершения фазы III артефакты остаются встроенными навсегда.  
- **После завершения фазы V** вызывается `setAltarBuiltInForge(true)` и вкладка переключается на плейсхолдер улучшений.  

*Документ самодостаточен для реализации.*
