# Изменения в системе бонусов к крафту

**Дата:** 2025-03-25  
**Версия:** 1.0

## Что изменилось?

### 1. Обновлена функция `calculateExpertiseImpact`

**Было:**
```typescript
export function calculateExpertiseImpact(expertise: number): ExpertiseImpact
```

**Стало:**
```typescript
export function calculateExpertiseImpact(material: MaterialNode, expertise: number): ExpertiseImpact
```

### 2. Изменён файл `src/types/materials/knowledge.ts`

- Добавлен импорт `MaterialNode` из `./material-core`
- Полностью переписана функция `calculateExpertiseImpact`
- Бонусы теперь рассчитываются на основе:
  - Экспертизы игрока (как и раньше)
  - Свойств материала (новое!)

### 3. Обновлён файл `src/components/encyclopedia/material-card.tsx`

- Изменён вызов функции:
  ```typescript
  // Было:
  const impact = calculateExpertiseImpact(expertise)
  
  // Стало:
  const impact = calculateExpertiseImpact(material, expertise)
  ```

## Новая логика бонусов

### 1. Скорость крафта
- **Экспертиза:** до 15% ускорения
- **Материал:**
  - Обрабатываемость (+35%)
  - Твёрдость (-20%)
  - Сложность переработки (-15%)
  - Теплопроводность (+30%)

### 2. Надёжность / Риск дефектов
- **Экспертиза:** до 50% снижения риска
- **Материал:**
  - Химическая стабильность (+25%)
  - Базовый риск дефектов (-25%)
  - Прочность на разрыв (+25%)
  - Потенциал чистоты (+25%)

### 3. Экономия материала
- **Экспертиза:** до 60% снижения отходов
- **Материал:**
  - Пористость (-30%)
  - Ремонтопригодность (+35%)
  - Обрабатываемость (+25%)
  - Прочность на разрыв (+10%)

### 4. Бонус к качеству
- **Экспертиза:** до +15
- **Материал:**
  - Твёрдость до +20
  - Прочность до +20
  - Магическая проводимость до +10
  - Редкость до +15

### 5. Точность прогноза
- **Экспертиза:** до +40% (от 50% до 90%)
- **Материал:**
  - Химическая стабильность до +20%
  - Магическая стабильность до +15%
  - Изученность до +15%

### 6. Разброс характеристик
- **Экспертиза:** до 80% снижения разброса
- **Материал:**
  - Химическая и магическая стабильность (35%)
  - Твёрдость + Прочность (35%)

## Примеры изменений

### Железо при экспертизе 50%

**Было (только экспертиза):**
- Скорость: -12.5%
- Надёжность: -35%
- Экономия: -40%
- Качество: +10
- Прогноз: 75%
- Разброс: 0.5

**Стало (экспертиза + материал):**
- Скорость: **-22%** (быстрее благодаря хорошей обрабатываемости)
- Надёжность: **-58%** (лучше благодаря низкому риску дефектов)
- Экономия: **-67%** (лучше благодаря высокой ремонтопригодности)
- Качество: **+26** (выше благодаря твёрдости и прочности)
- Прогноз: **76%** (почти одинаково)
- Разброс: **0.19** (намного меньше благодаря стабильности)

### Мифрил при экспертизе 50%

**Было (только экспертиза):**
- Скорость: -12.5%
- Надёжность: -35%
- Экономия: -40%
- Качество: +10
- Прогноз: 75%
- Разброс: 0.5

**Стало (экспертиза + материал):**
- Скорость: **-3%** (медленнее - сложен в обработке)
- Надёжность: **-65%** (лучше - очень стабильный)
- Экономия: **-52%** (средне)
- Качество: **+65** (гораздо выше - легендарный!)
- Прогноз: **86%** (очень точно)
- Разброс: **0.24** (низкий)

## Совместимость

- **Обратная совместимость:** Нет - сигнатура функции изменилась
- **Исправление:** Все вызовы `calculateExpertiseImpact` должны передавать `material` первым аргументом

## Тестирование

Для тестирования новой системы:

1. Откройте энциклопедию в игре
2. Используйте тестовую кнопку "Test: Iron 100%"
3. Сравните бонусы для разных материалов при одинаковой экспертизе
4. Обратите внимание на то, как разные свойства влияют на бонусы
