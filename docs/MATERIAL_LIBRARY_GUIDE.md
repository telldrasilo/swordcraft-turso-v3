# Шпаргалка по созданию материалов

**Назначение:** Быстрый и правильный способ добавлять новые материалы в проект

---

## Обзор структуры материала

### Полная структура MaterialNode

```typescript
interface MaterialNode {
  // 1. Идентификация
  identity: {
    id: string              // уникальный ID, латинскими буквами без пробелов
    name: string            // русское название, единственное число
    class: MaterialClass    // mineral | metal | wood | leather | other
    origin: MaterialOrigin  // natural | refined | alloy | composite | byproduct
    tags: string[]          // массив тегов (5-15 шт)
  }

  // 2. Физические свойства
  physical: {
    // Механические (0-200)
    density: number           // г/см³: лёгкие материалы (дерево) ~0.3-0.8, тяжёлые (металлы) ~6-9
    hardness: number          // твёрдость по условной шкале 0-200
    toughness: number         // ударная вязкость (прочность на разрыв) 0-200
    elasticity: number        // упругость (способность возвращать форму) 0-200

    // Тепловые
    meltingPoint: number      // температура плавления в °C (0-4000)
    ignitionPoint: number | null // температура воспламенения (null если не горит)
    thermalConductivity: number // условная теплопроводимость 0-100 (металлы ~50-90, дерево ~10-30)

    // Структурные
    porosity: number          // пористость в процентах 0-100 (дерево ~50-80, камень ~5-35)
    compressiveStrength: number // прочность на сжатие 0-200
    tensileStrength: number   // прочность на разрыв 0-200
  }

  // 3. Химические свойства
  chemical: {
    reactivity: number         // химическая активность 0-100
    stability: number          // устойчивость к воздействиям 0-100
    corrosionResistance: number // сопротивление коррозии 0-100
    oxidationResistance: number // сопротивление окислению 0-100
    acidity: number            // кислотность/щёлочность 0-14 (pH)
    solubility: number         // растворимость 0-100
  }

  // 4. Магические свойства
  arcane: {
    conductivity: number    // магическая проводимость 0-100
    affinity: number       // родство с магией 0-100
    stability: number      // устойчивость к магическим воздействиям 0-100
    resonance: number      // резонанс с определёнными школами магии 0-100
  }

  // 5. Технологические свойства
  processing: {
    workability: number      // обрабатываемость 0-100 (выше = легче)
    refineDifficulty: number // сложность переработки 0-100 (выше = сложнее)
    purityPotential: number  // максимальная чистота при идеальной обработке 0-100
    defectRisk: number       // риск дефектов при обычной обработке 0-100 (ниже = меньше риск)
    repairability: number    // ремонтопригодность 0-100 (для природных обычно 100, для сплавов может быть ниже)
  }

  // 6. Экономика
  economy: {
    rarity: number          // редкость 0-200 (выше = реже)
    tier: number           // уровень сложности 1-10
    baseValue: number      // базовая цена в золоте
    availability: number   // доступность для игрока 0-100
    discoverability: number // вероятность открытия 0-100
  }

  // 7. Резюме для UI
  summary: {
    basic: string           // 1-2 предложения, что это такое
    applied: string         // где используется (кузница/стройка/алхимия)
    strengths: string[]     // 2-5 плюсов
    weaknesses: string[]    // 1-3 минуса
    bestFor: string[]       // 2-5 лучших применений
  }

  // 8. Рецепт (если материал создаётся)
  recipe?: {
    process: ProcessType           // smelting | alloying | tanning | distillation | refining | crushing
    inputs: Array<{
      materialId: string
      quantity: number
      fraction: number  // доля в смеси (сумма должна быть 1.0)
    }>
    processModifiers?: {
      temperature?: number   // оптимальная температура
      duration?: number        // базовое время в секундах
      catalyst?: string | null
      qualityImpact?: number // множитель качества процесса (0.5-2.0)
    }
    yield: number             // выходной множитель (обычно 1.0, может быть меньше)
    byproducts?: Array<{
      materialId: string
      quantity: number
    }>
  }

  // 9. Пути открытия
  discovery: {
    unlockedBy: Array<{
      type: 'harvest' | 'craft' | 'use' | 'research' | 'mastery' | 'special'
      requiredExpertise: number  // минимальная экспертиза для открытия
      requirement?: {
        quantity?: number
        threshold?: number
        requiredMaterial?: string
        requiredBuilding?: string
      }
    }>
    researchCost?: number  // цена исследования в золоте
  }

  // 10. Остальное
  description?: string    // 1-3 абзаца, фоны, особенности
  icon?: string         // путь к иконке (начинается с /icons/resources/)
  version: number        // для миграций
}
```

## Типы и категории

### MaterialClass

| Класс | Описание | Примеры |
|--------|----------|----------|
| mineral | Природные минералы и руды | Железная руда, медная руда, кремень, гранит, обсидиан, кровавик |
| metal | Чистые металлы и сплавы | Железо, сталь, бронза, высокоуглеродистая сталь, серебряный сплав, холодное железо, мифрил |
| wood | Древесные материалы | Берёза, дуб, ясень, эбеновое дерево, железное дерево |
| leather | Кожаные материалы | Сырая кожа, выделанная кожа, бычья кожа, драконья кожа |
| other | Другие материалы | Кости, рога, янтарь, стекло, керамика, пластик |

### MaterialOrigin

| Происхождение | Описание | Примеры |
|---------------|----------|----------|
| natural | Природный, добытый из среды | Руды, дерево, кожа, камень |
| refined | Переработанный, очищенный от примесей | Слитки, обработанное дерево, выделанная кожа |
| alloy | Сплав двух или более компонентов | Бронза (медь+олово), сталь (железо+углерод), дуралюмин |
| composite | Композитный материал из разных слоёв | Деревянные композиты, слоистая бронза, углеволокно |
| byproduct | Побочный продукт другого процесса | Шлак, зола, шлаки, чугун (отход стали) |

## Шаги создания нового материала

### Шаг 1: Определение концепции

Ответьте на вопросы:

1. **Что это за материал?**
   - Базовый или улучшенный?
   - Особый или распространённый?
   - Есть ли реальные аналоги (исторические материалы)?

2. **Какие классы материалов он заменяет или дополняет?**
   - Создаёт конкуренцию?
   - Или заполняет пустую нишу?

3. **Какие сильные и слабые стороны?**
   - Материал должен иметь как минимум 1 минус
   - Плюсы должны быть уникальными или интересными
   - Например: "лёгкий но хрупкий", "очень твёрдый но дорогой"

4. **Какой редкий уровень?**
   - Сопоставьте с существующими материалами
   - common (0-40) — базовые материалы
   - uncommon (40-60) — улучшенные материалы
   - rare (60-100) — редкие материалы
   - epic (100-140) — эпические материалы
   - legendary (140-200) — легендарные материалы

5. **Какой уровень сложности (tier)?**
   - 1 — для новичков
   - 2 — для опытных кузнецов
   - 3 — для мастеров
   - 4-5 — для легендарных кузнецов

### Шаг 2: Выбор свойств

**Физические свойства (physical):**

| Свойство | Рекомендация | Как выбрать |
|-----------|--------------|--------------|
| density | Начните с диапазона для класса | Камни: 2.0-3.5 (плотные породы) |
| hardness | Сопоставьте с аналогами | Железная руда: ~45, бронза: ~65 |
| toughness | Прочность на разрыв | Кожа: низкая (30-40), камни: средняя (50-80) |
| elasticity | Упругость (способность гнуться) | Дерево: высокая (50-80), камни: низкая (10-30) |
| meltingPoint | Температура плавления | Металлы: ~1000-1500, руды: ~1000-1200 |
| ignitionPoint | Если горит — укажите | Дерево: ~300, уголь: ~400 |
| thermalConductivity | Теплопроводность | Металлы: ~50-80, дерево: ~10-30 |
| porosity | Пористость в % | Дерево: ~50-80, камни: ~5-35 |
| compressiveStrength | Прочность на сжатие | Камни: высокая (80-120), дерево: низкая (40-60) |
| tensileStrength | Прочность на разрыв | Камни: высокая (80-120), дерево: низкая (40-60) |

**Химические свойства (chemical):**

| Свойство | Рекомендация | Как выбрать |
|-----------|--------------|--------------|
| reactivity | Химическая активность | Металлы: средне (40-60), камни: низкие (20-40) |
| stability | Устойчивость | Металлы: средне (50-80), камни: высокие (70-90) |
| corrosionResistance | Сопротивление коррозии | Металлы: низкие (20-40), камни: средние (50-80) |
| oxidationResistance | Сопротивление окислению | Металлы: низкие (20-40), камни: высокие (70-90) |
| acidity | pH (0-14) | Металлы: нейтральные (6-8), дерево: слабокислые (4-6) |
| solubility | Растворимость | Металлы: низкие (5-15), дерево: нерастворимые (0-5) |

**Магические свойства (arcane):**

| Свойство | Рекомендация | Как выбрать |
|-----------|--------------|--------------|
| conductivity | Магическая проводимость | Металлы: низкие (5-20), кости: средние (20-40) |
| affinity | Родство с магией | Металлы: средние (20-40), драгоценные металлы: высокие (50-80) |
| stability | Устойчивость к магии | Камни: высокие (70-90), металл: средние (50-80) |
| resonance | Резонанс с определёнными школами | Камни: 0-20, кости: 10-30 |

**Технологические свойства (processing):**

| Свойство | Рекомендация | Как выбрать |
|-----------|--------------|--------------|
| workability | Обрабатываемость | Дерево: 80-100, камни: 0-50, кожа: 30-80 |
| refineDifficulty | Сложность переработки | Руды: 30-60, металлы: 40-80 |
| purityPotential | Максимальная чистота | Сплавы: 80-100, руды: 30-50, металл: 70-95 |
| defectRisk | Риск дефектов | Руды: 20-40, камни: 30-50, металл: 10-30 |
| repairability | Ремонтопригодность | Природные: 100, сплавы: 70-90 |

**Экономика (economy):**

| Свойство | Рекомендация | Как выбрать |
|-----------|--------------|--------------|
| rarity | Редкость (см. таблицу выше) | Основной баланс |
| tier | Уровень сложности (1-10) | Зависит от rarity |
| baseValue | Базовая цена в золоте | Базовые: 5-15, улучшенные: 15-30, редкие: 30-60 |
| availability | Доступность для игрока | Обычные: 90-100, редкие: 50-80 |
| discoverability | Вероятность открытия | Обычные: 100, редкие: 70-90 |

### Шаг 3: Создание резюме

**Суммарное описание (summary.basic):**
- 1-2 предложения, что это такое
- Например: "Природная железная руда. Требует переработки для получения чистого металла."

**Применение (summary.applied):**
- Где используется (одна или несколько систем)
- Например: "Сырьё для выплавки железа. Широко используется в кузнечном деле."

**Плюсы (summary.strengths):**
- 2-5 уникальных преимуществ
- Например: ["Широко распространена", "Низкая стоимость", "Основа металлургии"]

**Минусы (summary.weaknesses):**
- 1-3 недостатков
- Например: ["Требует переработки", "Низкая чистота", "Много примесей"]

**Лучшие применения (summary.bestFor):**
- 2-5 ситуаций, где материал показывает себя лучше всего
- Например: ["Выплавка железа", "Производство стали", "Базовое оружие"]

### Шаг 4: Путь открытия (discovery)

**Типы открытия:**
- harvest — добыча (автоматически открывается при первой добыче)
- craft — создание из других материалов (требует базовый материал)
- use — использование в других системах (строительство, алхимия)
- research — изучение в лаборатории (исследование свойств)
- mastery — мастерство (многократное использование для прокачки)
- special — особые события (квесты, открытия редких материалов)

**Рекомендации:**
1. **Базовые материалы:** Открывайте через harvest (requiredExpertise: 0)
2. **Руды:** Открывайте через harvest + craft (requiredExpertise: 5)
3. **Сплавы:** Открывайте через craft + research (requiredExpertise: 20)
4. **Редкие материалы:** Открывайте через research + special (requiredExpertise: 50)
5. **Легендарные материалы:** Открывайте через special (requiredExpertise: 75)

### Шаг 5: Рецепт (если нужен)

Если материал создаётся из других материалов:

**Типы процессов:**
- smelting — плавка руды в слиток
- alloying — создание сплава (два или более компонентов)
- tanning — обработка кожи
- refining — очистка материала от примесей
- distillation — дистилляция (для алхимии)
- crushing — дробление (для камней)

**Пример рецепта бронзы:**

```typescript
recipe: {
  process: 'alloying',
  inputs: [
    { materialId: 'copper_ingot', quantity: 9, fraction: 0.9 },
    { materialId: 'tin_ingot', quantity: 1, fraction: 0.1 },
  ],
  processModifiers: {
    temperature: 1100,
    duration: 300,
    qualityImpact: 1.1,
  },
  yield: 1.0,
}
```

### Шаг 6: Иконки

**Требования:**
- Формат: SVG или PNG
- Размер: 64x64 px
- Расположение: /icons/resources/
- Стиль: плоский, простой

**Советы:**
- Используйте одинаковый стиль для материалов одного класса
- Цветовая кодировка: металлы (серебро/медь), дерево (коричневый), камни (серый)

### Шаг 7: Описание (description)

**Требования:**
- 1-3 абзаца, не 5-10 предложений
- Фоны: история, геология, легендарные свойства, месторождения
- Тон: информативный, но не скучный

**Пример:**
```
Железная руда — основной источник железа. Встречается в шахтах и горных районах. 
Содержит оксиды железа с примесями породы. Требует выплавки в печи 
для получения металла. Широко используется в кузнечном деле и является основой 
металлургии.
```

## Структура файлов (Рекомендация)

Рекомендуется следующая структура для удобства:

```
src/data/materials/
├── library/                    # Отдельные файлы для каждого материала
│   ├── iron-ore.ts         # Железная руда
│   ├── copper-ore.ts        # Медная руда
│   ├── tin-ore.ts           # Оловянная руда
│   ├── iron-ingot.ts        # Железный слиток
│   ├── copper-ingot.ts      # Медный слиток
│   ├── bronze-ingot.ts       # Бронза
│   └── ...
│
├── collections/                 # Коллекционные файлы (по необходимости)
│   ├── ores.ts              # Все руды (для удобства поиска)
│   ├── ingots.ts            # Все слитки
│   ├── stones.ts            # Все камни
│   ├── wood.ts              # Все древесные материалы
│   └── leathers.ts          # Все кожаные материалы
│
└── index.ts                 # Главный файл, который экспортирует все материалы
```

### Плюсы такой структуры:

1. **Масштабируемость** — легко добавлять новые материалы без конфликтов
2. **Удобство поиска** — коллекционные файлы облегчают навигацию
3. **Чистота** — каждый материал в отдельном файле
4. **Организованность** — логически сгруппированные материалы

### Минусы такой структуры:

1. **Больше файлов** — при 10-100 материалов будет 10-20 файлов в директории library/
2. **Дублирование** — нужно поддерживать синхронизацию с коллекционными файлами
3. **Больше импортов** — больше `import` для каждого файла

## Контрольный список перед созданием

Перед созданием нового материала убедитесь, что:

### 1. Уникальность ID
- [ ] ID уникален (нет дубликатов)
- [ ] ID использует snake_case (нижний регистр с подчёркиваниями)
- [ ] ID соответствует названию (например, iron_ore, copper_ingot)

### 2. Баланс свойств
- [ ] Свойства находятся в рекомендуемых диапазонах
- [ ] Есть хотя бы один минус
- [ ] Минусы не дублируют плюсы
- [ ] Редкость соответствует сложности материала

### 3. Логичность свойств
- [ ] Физические свойства логичны (например, твёрдый материал не имеет низкую твёрдость)
- [ ] Обрабатываемость соответствует происхождению (природные труднее обрабатывать)
- [ ] Магические свойства соответствуют классу (металлы низкая проводимость)

### 4. Резюме
- [ ] Краткое и информативное (не 10 предложений)
- [ ] Применение указано (не пустое)
- [ ] Есть 2-5 плюсов и 1-3 минуса
- [ ] Лучшие применения релевантны

### 5. Пути открытия
- [ ] Есть хотя бы один способ открытия
- [ ] Значения требуемой экспертизы реалистичны
- [ ] Стоимость исследования (если есть) адекватна

### 6. Иконка
- [ ] Указан путь к иконке
- [ ] Иконка существует (добавьте в /icons/resources/)
- [ ] Размер 64x64 px

### 7. Описание
- [ ] 1-3 абзаца
- [ ] Информативно
- [ ] Без орфографических ошибок

### 8. Формат
- [ ] Использован правильный TypeScript-формат
- [ ] Все поля заполнены
- [ ] Нет лишних запятых

### 9. Интеграция
- [ ] Материал экспортируется в index.ts
- [ ] Категория отображается корректно
- [ ] Редкость и тир отображаются правильно

## Советы по балансу

### Редкие материалы
Не создавайте много редких материалов сразу. Редкость должна быть заработана:
- Больше времени на исследования
- Более высокие цены
- Сложнее найти или добыть

### Легендарные материалы
Легендарные материалы должны быть особенными:
- Очень уникальные свойства (не просто +10% ко всему)
- Особый способ открытия (квест, открытие легендарного месторождения)
- Связаны с сюжетом или лором

### Бонусы и минусы
Избегайте:
- Материалов без минусов (они должны быть лучшими во всём)
- Материалов с минусами, которые не влияют на геймплей (незначительные штрафы)
- Чрезмерно специализированные материалы (например, только для одного вида оружия)

### Свойства
Избегайте:
- Свойства, которые не используются ни в одной системе
- Нереалистичные значения (например, плотность камня = 0.1 или 20.0)
- Свойства, которые противоречат физике реального мира

## Вопросы для уточнения

Перед созданием нового материала уточните:

1. **Нужен ли рецепт?**
   - Если да — какие входные материалы?
   - Какой тип процесса (smelting/alloying/etc)?

2. **Какие особые способности?**
   - Магические свойства выше обычных?
   - Особые технологии обработки?
   - Связь с определёнными школами магии?

3. **Какой уровень открытия?**
   - Какое минимальное значение экспертизы?
   - Сколько стоит исследование (если есть)?

4. **Нужны ли побочные продукты?**
   - Отходы производства (шлак, зола)?
   - Сопряжённые материалы?
