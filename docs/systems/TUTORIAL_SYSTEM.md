# Система туториала

## Обзор
Туториал помогает игроку пройти по основным системам игры и объясняет базовый цикл прогрессии.

Главный файл:
- `src/store/slices/tutorial-slice.ts`
- `src/data/tutorial.ts`

## Состояние
Слайс хранит:
- `isActive`
- `currentStep`
- `completedSteps`
- `skipped`

## Основные действия
- `startTutorial()`
- `completeCurrentStep()`
- `skipTutorial()`
- `resetTutorial()`

## Назначение шагов
Обычно шаги ведут игрока через:
- ресурсы
- кузницу
- создание первого оружия
- наем работников
- заказы или экспедиции
- магические системы

## Связи с другими системами
Туториал не изолирован: он проверяет прогресс игрока в основных системах и подталкивает к ключевым действиям.

## Что смотреть при изменениях
- `src/store/slices/tutorial-slice.ts`
- `src/data/tutorial.ts`
