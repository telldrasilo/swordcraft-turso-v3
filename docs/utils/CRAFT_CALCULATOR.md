# Калькулятор крафта

## Обзор
Этот документ описывает вычислительное ядро кузницы. Главная точка входа — `src/lib/craft/calculator.ts`, а прогноз до запуска крафта строится через `src/lib/craft/forecast-calculator.ts`.

## Главные файлы
- `src/lib/craft/calculator.ts` — итоговые характеристики после крафта
- `src/lib/craft/forecast-calculator.ts` — прогноз диапазонов до крафта
- `src/lib/craft/constants.ts` — **именованные коэффициенты**: расчёт оружия, прогноз, умная сортировка материалов (`material-sorting.ts`)
- `src/lib/craft/formulas.ts` — чистые хелперы (`applyPercentMultiplier`, `contributionFromMaterialPercent`)
- `src/lib/craft/material-sorting.ts` — оценки для селектора материалов
- `src/lib/store-utils/craft-utils.ts`
- `src/types/craft-v2.ts`
- `src/types/forecast.ts`

При изменении баланса крафта правьте прежде всего **`constants.ts`**, затем при необходимости формулы в `calculator` / `forecast-calculator` и обновите тесты в `src/lib/craft/*.test.ts`.

## Что делает `calculateWeapon()`
Функция собирает итоговое оружие из:
- рецепта
- набора материалов по частям
- выбранных техник
- систем качества и бонусов

На выходе формируются:
- итоговые статы
- качество
- grade качества
- множитель качества
- цена продажи
- служебная детализация для UI

## Основные шаги расчета
1. Берется рецепт и его базовые характеристики.
2. Для каждой части оружия считываются назначенные материалы.
3. Материалы преобразуются в набор вкладов в статы.
4. Поверх накладываются техники.
5. Рассчитывается итоговое качество.
6. К качеству применяется множитель.
7. Вычисляется цена продажи и допполя результата.

## Источники влияния
На расчет влияют:
- базовые статы рецепта
- свойства материалов
- редкость материалов
- quality bonuses материалов
- техники
- экспертиза по материалам
- ограничения рецепта по частям

## Качество
Используются функции из `src/types/shared/quality.ts`:
- `getQualityGrade()`
- `getQualityMultiplier()`
- `getQualityNameRu()`

Качество участвует минимум в трех местах:
- формирует читаемую grade-метку
- масштабирует характеристики
- влияет на цену и ценность оружия

## Прогноз `calculateForecast()`
Прогноз строится до старта крафта и показывает диапазоны.

Он нужен для:
- UI предпросмотра
- оценки риска
- сравнения вариантов материалов
- подсказок по точности

Прогноз возвращает:
- ожидаемое качество
- quality range
- диапазоны характеристик
- confidence
- accuracy

## Диапазоны и точность
Диапазоны зависят от:
- экспертизы по материалам
- сложности сборки
- разброса бонусов материалов
- неопределенности от техник и комбинаций

Чем выше знание материалов, тем:
- уже диапазоны
- выше confidence
- меньше расхождение между прогнозом и фактом

## Важные формулы на уровне системы
В коде используются вычисления следующих типов:
- суммирование вкладов материалов по частям
- нормализация вкладов по весам
- применение множителей качества
- расчет sell price на основе итоговой силы предмета
