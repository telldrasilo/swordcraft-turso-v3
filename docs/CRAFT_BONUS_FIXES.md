# Исправления в системе бонусов к крафту

**Дата:** 2025-03-25  
**Версия:** 1.1

## Проблемы и исправления

### 1. Исправлена формула для "Экономия" (materialWasteMultiplier)

**Проблема:** Все материалы показывали -90% экономии независимо от свойств.

**Причина:** В формуле все знаки были минус, что делало результат некорректным.

**Стало:** Все знаки заменены на плюс, так как все свойства уменьшают отходы.

### 2. Исправлена формула для "Скорость" (timeMultiplier)

**Проблема:** Знаки в формуле были некорректными.

**Стало:** Явные знаки в коэффициентах (например, -0.20 для твёрдости).

### 3. Исправлена формула для "Надёжность" (defectRiskMultiplier)

**Проблема:** Знак при defectRisk был неправильным.

**Стало:** Явный минус (-0.25) для defectRisk.

### 4. Обновлена тестовая кнопка

**Было:** Кнопка устанавливала экспертизу только для железа.

**Стало:** Кнопка циклически переключает экспертизу для ВСЕХ отображаемых материалов:
1. "Test: All 100%" → устанавливает 100% всем материалам
2. "Test: All 10%" → устанавливает 10% всем материалам
3. "Test: Reset" → сбрасывает в нормальный режим
