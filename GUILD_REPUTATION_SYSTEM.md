# Система репутации гильдии

## Обзор

Система репутации гильдии предоставляет долгосрочный прогресс для игрока через выполнение заказов на крафт, экспедиций и подземелий. Чем выше уровень гильдии, тем больше бонусов получает игрок.

---

## Начисление репутации

### Формула

```
reputation = baseReward × activityMultiplier × playerLevelMultiplier
```

Где:
- `baseReward` — базовая награда (goldReward для заказов, baseGold для экспедиций)
- `activityMultiplier` — множитель по типу деятельности (см. ниже)
- `playerLevelMultiplier` — множитель уровня кузнеца (стимулирует прокачку)

### Множители по типу деятельности

| Тип деятельности | Множитель | Описание |
|----------------|-----------|----------|
| Крафт (заказы) | 1.0× | Базовое начисление |
| Экспедиции | 1.5× | Экспедиции дают в 1.5 раза больше репутации |
| Подземелья | 1.2× | Подземелья дают в 1.2 раза больше (заглушка) |

### Множитель уровня кузнеца

```
playerLevelMultiplier = 0.5 + playerLevel × 0.05
```

Примеры:
- Уровень 1: `0.5 + 1 × 0.05 = 0.55`
- Уровень 5: `0.5 + 5 × 0.05 = 0.75`
- Уровень 10: `0.5 + 10 × 0.05 = 1.0`
- Уровень 20: `0.5 + 20 × 0.05 = 1.5`

Это означает, что кузнец уровня 20 получает в 3 раза больше репутации за ту же деятельность, чем кузнец уровня 1.

### Примеры расчёта

#### Пример 1: Заказ на крафт (уровень кузнеца 3)
- Золотая награда: 50
- Множитель кузнеца: 0.65
- Множитель активности: 1.0
- Итого: `50 × 1.0 × 0.65 = 32` репутации

#### Пример 2: Экспедиция (уровень кузнеца 5)
- baseGold: 120
- Множитель кузнеца: 0.75
- Множитель активности: 1.5
- Итого: `120 × 1.5 × 0.75 = 135` репутации

## Влияние уровня гильдии

### 1. Количество активных экспедиций

Уровни гильдии открывают дополнительные слоты для экспедиций:

- **Уровень 1:** 1 экспедиция
- **Уровень 3:** 2 экспедиции
- **Уровень 10:** 3 экспедиции

Проверка при запуске:
```typescript
const maxActive = getMaxActiveExpeditions(guild.level)
if (guild.activeExpeditions.length >= maxActive) {
  return false // Нельзя запустить
}
```

### 2. Максимум известных искателей

Уровень гильдии влияет на лимит базы известных искателей:

- Уровень 1: 5 искателей
- Уровень 10: 35 искателей

Это напрямую влияет на количество контрактов, которые можно заключить.

### 3. Разблокировка экспедиций

Каждая экспедиция имеет свойство `minGuildLevel`. Экспедиции открываются постепенно по мере роста уровня гильдии:

- Уровень 1: Лёгкие экспедиции (easy)
- Уровень 2: Обычные экспедиции (normal)
- Уровень 3-5: Сложные экспедиции (hard)
- Уровень 6-8: Экстремальные экспедиции (extreme)
- Уровень 10: Легендарные экспедиции (legendary)

## Использование в коде

### Расчёт репутации

```typescript
import { calculateReputationGain } from '@/types/guild'

// За заказ на крафт
const reputation = calculateReputationGain(
  'craft',        // Тип деятельности
  50,            // Золотая награда
  playerLevel     // Уровень кузнеца
)

// За экспедицию
const reputation = calculateReputationGain(
  'expedition',
  120,           // baseGold из шаблона экспедиции
  playerLevel
)
```

### Добавление репутации

```typescript
import { useGameStore } from '@/store/game-store-composed'

const store = useGameStore()

// Автоматически повышает уровень гильдии
store.addReputation(amount)
```

### Получение информации об уровне

```typescript
import {
  getGuildReputationLevel,
  getReputationToNextLevel,
  getMaxActiveExpeditions,
} from '@/types/guild'

const level = getGuildReputationLevel(reputation)
const toNext = getReputationToNextLevel(reputation, level)
const maxExpeditions = getMaxActiveExpeditions(level)
```

## Стратегии прокачки

### Начало игры (уровни 1-3)
- **Фокус:** Заказы на крафт
- **Почему:** Быстрее, проще, требуют меньше ресурсов
- **Цель:** Быстро достичь уровня 3 для второго слота экспедиций

### Середина игры (уровни 3-7)
- **Фокус:** Экспедиции
- **Почему:** Дают больше репутации, дополнительно награждают War Soul для оружия
- **Цель:** Разблокировать сложные экспедиции и увеличить базу искателей

### Поздняя игра (уровни 8-10)
- **Фокус:** Сложные и легендарные экспедиции
- **Почему:** Максимальная репутация за одну экспедицию
- **Цель:** Достичь 3 слотов экспедиций и разблокировать весь контент

## Баланс и будущее

### Текущий баланс
- Крафт: ~25-50 репутации за заказ (в зависимости от уровня)
- Лёгкие экспедиции: ~30-60 репутации
- Сложные экспедиции: ~100-200 репутации
- Легендарные экспедиции: ~300-600 репутации

### Планируемые улучшения
- **Подземелья:** Реализовать механику с начислением репутации (1.2×)
- **Специальные задания:** Ограниченные по времени задания с бонусной репутацией
- **События гильдии:** Глобальные события, влияющие на репутацию
- **Достижения:** Бонусная репутация за достижение вех

## Заключение

Система репутации гильдии предоставляет мотивацию для долгосрочной игры, связывая все системы (крафт, экспедиции, подземелья) в единый прогресс. Экспоненциальное требование репутации для более высоких уровней создаёт долгосрочные цели и поддерживает интерес к игре.
